Order in which CSS applies

  • body (everything in Body)
  • Class (Class overides body)
  • Class Order (Lower the order, the more priority)
  • ID (ID more specific than classes, and dosent matter order)
  • Inline (Overrides all above)
  • !important will overide everything (most specific)
